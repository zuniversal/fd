字符串类

一系列UTF-16代码单元。

字符串主要用于表示文本。字符可以由多个代码点表示，每个代码点由一个或两个代码单元组成。例如，巴布亚新几内亚国旗角色需要四个代码单元来表示两个代码点，但应该被视为单个字符：“🇵🇬”。不支持标志字符的平台可能会显示字母“PG”。如果代码点被交换，则它变为瓜德罗普岛标志“🇬🇵”（“GP”）。

字符串可以是单行或多行。单行字符串使用匹配的单引号或双引号编写，多行字符串使用三引号编写。以下是所有有效的Dart字符串：

'Single quotes';
"Double quotes";
'Double quotes in "single" quotes';
"Single quotes in 'double' quotes";

'''A
multiline
string''';

"""
Another
multiline
string""";

字符串是不可变的。虽然您无法更改字符串，但您可以对字符串执行操作并将结果分配给新字符串：

var string = 'Dart is fun';
var newString = string.substring(0, 5);

您可以使用plus（+）运算符来连接字符串：

'Dart ' + 'is ' + 'fun!'; // 'Dart is fun!'

您还可以使用相邻的字符串文字进行连接：

'Dart ' 'is ' 'fun!';    // 'Dart is fun!'

您可以使用${}在字符串中插入Dart表达式的值。评估标识符时可以省略花括号：

string = 'dartlang';
'$string has ${string.length} letters'; // 'dartlang has 8 letters'

字符串由可通过codeUnitAt或codeUnits成员访问的Unicode UTF-16代码单元序列表示：

string = 'Dart';
string.codeUnitAt(0); // 68
string.codeUnits;     // [68, 97, 114, 116]

代码单元的字符串表示形式可通过索引运算符访问：

string[0];            // 'D'

字符串的字符以UTF-16编码。解码组合代理对的UTF-16产生Unicode代码点。遵循与Go类似的术语，我们使用名称'rune'表示Unicode代码点的整数。使用runes属性获取字符串的符文：

string.runes.toList(); // [68, 97, 114, 116]

对于由代理对组成的基本多语言平面（平面0）之外的字符，符文组合该对并返回单个整数。例如，符号值为0x1D11E的音乐G-clef（'𝄞'）的Unicode字符由UTF-16代理对组成：0xD834和0xDD1E。使用codeUnits返回代理项对，并使用runes返回它们的组合值：

var clef = '\u{1D11E}';
clef.codeUnits;         // [0xD834, 0xDD1E]
clef.runes.toList();    // [0x1D11E]

无法扩展或实现String类。尝试这样做会产生编译时错误。
